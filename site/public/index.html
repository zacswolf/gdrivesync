VS Code extension companion site

Google files to Markdown or CSV, with a real sync button.

GDriveSync links Google Docs and DOCX files to normal Markdown files, and Google Sheets and XLSX files to local CSV output in VS Code. Sync is one-way by design: Google Drive stays the source of truth.

Why this site exists

The extension uses this domain for its public homepage, privacy policy, Google OAuth bridge, and hosted Google Picker flow for supported Docs, Sheets, DOCX, and XLSX sources.

What the extension stores

OAuth tokens stay in VS Code secret storage. File mappings and sync metadata live in a workspace-side .gdocsync.json manifest, while generated Markdown assets and spreadsheet CSV folders stay in your workspace.

Privacy

The extension only uses Google Drive access to read supported Google files that you explicitly link. Read the full policy on the privacy page.
