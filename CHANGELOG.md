# Changelog

## 0.1.0

- Initial public-ready feature set for GDriveSync.
- Google Docs and DOCX -> Markdown sync with extracted sibling asset folders
- Google Slides and PowerPoint -> Marp-flavored Markdown sync
- Google Sheets and XLSX -> CSV sync with automatic single-file or folder output switching
- Multi-account Google auth in both the VS Code extension and CLI
- Agent-friendly CLI with `auth`, `inspect`, `export`, `link`, `status`, `sync`, `unlink`, and `doctor`
- Optional image enrichment for Markdown and Marp outputs using local OCR, OpenAI, or Anthropic
- Cloudflare Pages hosted site for homepage, privacy policy, OAuth bridge, and hosted Picker fallback
